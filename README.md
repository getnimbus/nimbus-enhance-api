# Nimbus Enhance API

Build for Nodereal Hackathon

## How to use

1. Run `make` to build executable file
2. Add `.env` file

```env
ENV=local
DEBUG=no
REDIS_ADDRESS=localhost:6379
REDIS_DB=0
REDIS_USER=
REDIS_PASS=
```

3. Run executable file
